# 📊 Student Marks Analysis Using Python

## 📌 Overview

This project analyzes student performance using Python. It loads student marks from a CSV dataset, performs data analysis using Pandas and NumPy, and visualizes the results using Matplotlib.

## 🚀 Features

- Load student data from a CSV file
- Analyze student marks
- Calculate average, maximum, and minimum marks
- Identify high-scoring students
- Display the top-performing student
- Generate histogram and bar chart visualizations

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## 📂 Project Structure

```
Student-Marks-Analysis/
```

## ▶️ How to Run

1. Install the required libraries:

```bash
pip install pandas numpy matplotlib
```

2. Run the project:

```bash
python Project.py
```

## 📈 Output

- Histogram of Marks Distribution
- Bar Graph of Average Marks vs Number of Courses
- Student Statistics
- High-Scoring Students
